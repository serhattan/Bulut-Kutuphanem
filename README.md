<h1>Bulut-Kütüphanem</h1>

<b>To working with this project:</b>

1) You need to set up nodejs in your computer

2) Open the Terminal(command prompt) and go to directory of your nodejs file. It's look like C:\Program Files\nodejs>

3) Now you shold execute the <code>npm install -g cordova@6.0.0)</code> and then <code>npm install -g ionic</code> command. (If occur an error You can add <code>sudo</code> into your command to solve problem)

4) After installing successfully completed open the Terminal and go to the directory mycloudlibrary inside of Bulut-Kutuphanem folder.

5) And finally you should execute <code>ionic serve</code> command.
